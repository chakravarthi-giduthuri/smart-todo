# App Assets

Place the following files here before building:

- icon.png — 1024x1024px PNG, no alpha, orange background (#ec5b13) with white checkmark
- splash.png — 1284x2778px PNG for splash screen
- adaptive-icon.png — 1024x1024px PNG for Android adaptive icon (foreground only, transparent bg)
- favicon.png — 48x48px for web

Run: npx expo-image-optimizer to optimize after adding
